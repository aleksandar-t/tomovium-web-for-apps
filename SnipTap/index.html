How to use SnipTap

A quick screen-to-screen guide for the main workflows.

Create a snippet

Tap the plus button, enter a title, choose a category, write the content, and save.

Copy a saved snippet

Open the Home screen and tap a snippet card to copy its content instantly.

Use a template

Create a snippet with placeholders like [name] or [date]. SnipTap will open a fill screen before copying.

Save from clipboard

Open Clipboard, preview copied text, then use Save to turn it into a snippet.

Rewrite with AI

Open a saved snippet and use AI Rewrite to polish the text before copying or saving a new version.

Use the floating bubble

Enable the bubble to quickly open SnipTap, open Clipboard, or create a snippet from copied text.

Back up your data

Use Import / Export to export your snippets. Import is available with SnipTap Pro.
